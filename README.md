# DS340Project

This is the Github repository for the "Detecting Market Structure Using Long-Short
Term Memory Neural Networks" project completed during the "DS340: Introduction to Machine Learning and AI" course at Boston University's Faculty of Computing & Data Sciences in the Fall 2025 semster, authored by Yulian Gogov and Sviatoslav Shevchenko. Above you can see the various project components sorted into relevant folders, below will be a short summary of instructions on how to reproduce the project. 


The project has fundamentally two parts: Technical and Sentiment, which first start separately and eventually merge later on before the construction of the final LSTM.
